# Personal Portfolio — React

My personal portfolio website built with React, TypeScript, and Vite.

## Sections

- **Home** — Hero section and credibility strip
- **About** — Brief personal introduction
- **Skills** — Technologies and tools
- **Experience** — Work history timeline
- **Contact** — Contact form / links

## Tech Stack

- [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [SCSS](https://sass-lang.com/)
- [ESLint](https://eslint.org/) + [Prettier](https://prettier.io/) + [Husky](https://typicode.github.io/husky/) (pre-commit hooks)

## Getting Started

```bash
npm install
npm start        # dev server  →  http://localhost:5173
```

## Scripts

| Command | Description |
|---|---|
| `npm start` | Start the development server |
| `npm run build` | Type-check and build for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint |
| `npm run prettier:check` | Check formatting |
| `npm run prettier:format` | Auto-format source files |
